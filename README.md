# FinCal Retirement Planning Calculator

Built for FinCal Innovation Hackathon – Technex'26

## Team
Soumya Dhole  
Shashwat Deshpande

## Tech Stack
Frontend: Next.js  
Backend: Node.js + Express  

## Features
- Inflation adjusted future expenses
- Retirement corpus calculation
- Monthly SIP required
- Retirement withdrawal simulation
- Interactive financial graphs

## How to Run

### Start Backend
cd backend  
node server.js

### Start Frontend
cd frontend  
npm run dev
